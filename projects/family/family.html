Responsive Pages

Why is the viewport meta tag important? - It ensures proper scaling and responsiveness on different devices, especially mobile.

How can you adjust the CSS for printing? - Use the print media query, e.g., '@media print { /* styles */ }' or 'link rel="stylesheet" href="print.css" media="print">'

When would you want to adjust the CSS for printing? - To optimize layout, hide unnecessary elements, and fit content to paper size.

Why do we need a viewport meta tag? - Viewport meta tag allows responsive design to be applied to the size of the device screen.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? - The style declared later applies. However, if the first declaration has an important, it will be applied first.

How do you change the CSS for a web page displayed on a phone? - Use Media Queries and this applies to style when screen width(max width) is less than 600px.
