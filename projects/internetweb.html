Internet

Internet is a global computer network giving a variety of information and communication facilities.

Web

The web is a collection of public web pages that are hyperlinked and accessible through the Internet

How are the Internet and Web different?

The Internet is a global computer network that connects to other networks, and the web is a service that runs on the Internet. It also consists of documents and resources that are hyperlinked.

ASCII and Unicode

Both ASCII and Unicode are character encoding standards that represent text on computers and other devices. ASCII is the old standard that supports a limited number of characters, while Unicode is the latest standard that supports more characters and symbols. ASCII also uses one byte (8 bits) to represent each character.

What is UTF-8

UTF-8 is a character encoding system that uses one to four bytes to represent characters. It is the most common character encoding on the world wide web.

URL

URL is a resource address on the Internet, such as an HTML page. A URL is used in browsers to find and retrieve resources from the web.

Domain Name

A domain name is a text string that identifies a website or email address on the Internet. It is a human-readable alternative to the complex IP addresses that computers use to find websites.

HTTP

HTTP is a set of rules that regulate how web browsers and web servers exchange data over the Internet. The rules are the foundation of the world wide web and are used to load web pages.

HTTPS

HTTPS is a protocol that encodes the communication between a user's browser and a website. Also, it is the secure version of HTTP

IP Address

IP address is a unique number that identifies devices connected to the Internet or a local network. It is a numeric label assigned to a device that communicates using the Internet.

DNS

Domain Name System (DNS) converts domain names to IP addresses and uses them to load Internet pages by browsers.

W3C Recommendations

A W3C Recommendation is a set of guidelines or specifications that set technical standards for web technology.

XML,XHTML,HTML,HTML5

XML is a programming language that allows users to store, specify, and share database between computer systems. XHTML is markup language that integrates the structure of HTML with the pliability of XML. HTML is a text-based computer language that describes the structure and content of web pages. HTML5 is a markup language used for structuring and describing hypertext documents on the World Wide Web.
