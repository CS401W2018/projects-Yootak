Explain the difference between inline, embedded, and external CSS.

:To create an image, use color (property & value), font-size, font-family, and so on. The Inline CSS - style attribute is applied directly to individual HTML elements using attributes. The Embedded CSS - style tag is written within the tags inside the head section style of the HTML file. The External CSS-file is written as a separate .css file and is associated with the HTML document using the link tag.

Explain why you should only use external CSS.

: The reason why I must use external CSS when working on a big project is that I have a hard time keeping and managing styles internally. Also, external CSS helps you maintain a consistent structure by reusing styles on multiple pages.
