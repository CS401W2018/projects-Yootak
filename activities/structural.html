Structual Tags

What goes in header?

The header contains website logos, site titles, and search windows.

What goes in the Nav? Can the Nav go in the Header? Does it have to go in the header? Does the Nav contain links to another website?

Nav mainly includes home, service, introduction, and contacts. Yes, you can put the header in Nav to align navigation to the website's branding and introduction. The header does not have to be in Nav. It can also be shown in footer tag. Nav was built primarily for internal links, but can contain external links.

What goes in Main?

The main page contains the core content of the page. Each page should contain unique content, such as a product description or service overview.

What goes in the Footer?

Footer includes copyright information, contacts, and map links used for information at the bottom of the web page.
