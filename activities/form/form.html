What is AJAX? AJAX is a technique that enables web pages to communicate with servers without requiring a page reload.User can process data and update sections of the page without reloading the full page when submit forms using AJAX.

Why do you put your JS Code in a separate file? Because it keeps HTML cleaner by keeping structure, style, and behavior distinct, JavaScript code is stored in a separate file. Additionally, JS allows for easy code updates and the reuse of the same script across multiple sites.
